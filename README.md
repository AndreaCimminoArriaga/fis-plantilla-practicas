# fis-plantilla-practicas
Una plantilla basada en Java spark para el desarrollo de un servicio Web siguiendo el paradigma MVC.

#### Consideraciones
 * La plantilla utiliza Java 1.8
 * No cambiar el nombre de la clase `App`
